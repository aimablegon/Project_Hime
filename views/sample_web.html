UI/UX Design Publishing Practice

웹디자이너 · 웹퍼블리셔의 역할은 디자인과 코딩 두가지 영역으로 나뉘며, 그래픽툴을 이용하여 웹 및 모바일의 표준에 맞추어 디자인하는 작업을 말합니다. 단순히 심미적인 것에 초점을 맞추는 것이 아니라 웹이라는 공간의 특성 및 기능을 파악하여 사용자가 보다 쉽고 편리하게 사이트를 이용할 수 있도록 UI(사용자의 환경)/UX(사용자의 경험)을 고려하여 디자인해야합니다. 웹퍼블리셔는, 웹표준, 웹접근성 전문가로서, 구조기술, 표현기술, 호환기술을 두루 갖춘 프론트엔드 개발을 수행합니다.

UI User Interface UX User Experience

하이미디어의 웹디자인·웹퍼블리싱 과정은,

웹전문가로서 도약할 수 있도록 창의적인 아이디어를 시각화 하는데 필요한 수업과, 이를 웹이라는 특성에 맞게 퍼블리싱 하는 능력을 두루 학습하는 실무위주의 프로젝트 수업으로서, 본 과정을 통해 우수한 포트폴리오를 가지고 이미 많은 산업체에 취업을 하여 실력을 입증받고, 대부분의 학생들이 산업체에서 인정받는 당당한 웹디자이너와 웹퍼블리셔로 성장하고 있습니다.
